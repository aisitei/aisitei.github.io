주요 내용

OpenAI가 AI 비디오 생성 서비스인 Sora의 운영을 중단하기로 결정했습니다. 이는 2026년 상반기 예정된 기업공개(IPO)를 앞두고 상품 포트폴리오를 간소화하기 위한 전략적 결정으로 보입니다. Sora는 2025년 9월 말 출시되었으나, Apple App Store에서 일시적으로 1위를 차지한 후 사용자 관심도가 급격히 떨어졌습니다.

서비스 종료 영향

🎥

개발자 API 중단

Sora 개발자 API 버전도 함께 중단되어 관련 서비스들이 영향을 받습니다.

💬

ChatGPT 비디오 기능 제거

ChatGPT에서 제공되던 비디오 생성 및 편집 기능이 완전히 제거됩니다.

🏢

Disney 파트너십 종료

3년간의 Disney 협력 관계가 종료되며 Star Wars, Pixar, Marvel 콘텐츠 관련 계획이 취소됩니다.

📹

비디오 모델 기술 단계적 폐기

CEO Sam Altman이 밝힌 바에 따르면, 비디오 모델 기술을 사용하는 모든 제품을 점진적으로 중단할 예정입니다.

상세 영향 분석

🎬 개발자 및 서비스 제공자

Sora API를 기반으로 구축된 모든 애플리케이션과 서비스가 영향을 받게 됩니다:

  • API 버전 완전 중단
  • 기존 통합 서비스 폐기
  • 개발자 지원 종료

🤖 ChatGPT 사용자

ChatGPT 플랫폼 사용자들이 이용하던 비디오 기능이 제거됩니다:

  • AI 비디오 생성 기능 삭제
  • 비디오 편집 도구 제거
  • 관련 플러그인 지원 중단

🎞️ Disney 미디어 파트너십

가장 큰 영향을 받는 부분은 Disney와의 협력 관계 종료입니다:

  • 예상 10억 달러 규모의 투자 및 라이선싱 계약 취소
  • Star Wars, Pixar, Marvel 콘텐츠 제휴 종료
  • Disney 개발팀의 Sora 개발자와의 협력 중단
  • 발표 직전날 밤까지 협력 중이던 프로젝트 전면 취소

📺 전체 비디오 모델 기술 단계적 폐기

Sam Altman CEO의 공식 입장:

  • 비디오 모델 기술을 사용하는 모든 제품 단계적 중단
  • 향후 비디오 기반 AI 서비스 개발 계획 없음
  • 리소스 재배치로 기업공개 준비에 집중

서비스 종료 배경

💼 공개 기업 기준 충족 필요성

OpenAI의 CFO는 "우리는 공개 기업 기준을 충족하는 회사를 구축해야 한다"는 입장을 표명했습니다. 이는 다음을 의미합니다:

  • 엔터프라이즈급 제품 우선순위 상향
  • 소비자 대상 애플리케이션 제거
  • 수익성 있는 사업 포트폴리오 구성

📊 포트폴리오 정렬

OpenAI는 IPO 준비 과정에서 상품 포트폴리오를 간소화하고 있습니다:

  • 핵심 비즈니스에 집중 (API, 엔터프라이즈 솔루션)
  • 수익 창출이 어려운 소비자 상품 폐지
  • 장기 전략적 가치가 낮은 파트너십 종료

⚠️ 기업공개 관련 위험 요소

OpenAI의 IPO 신청서에 "Microsoft에 대한 의존도"가 잠재적 위험 요소로 명시되어 있습니다:

  • Microsoft와의 파트너십 과도 집중
  • 인프라 및 자본 의존성
  • 기술 로드맵의 외부 의존성
