Claude가 컴퓨터 제어를 테스트합니다

출처: IT Home | 2026년 3월 25일

주요 소식

🚀 Anthropic의 새로운 기능

Anthropic은 유료 Claude Pro 및 Claude Max 구독자를 위해 "Computer Use" 미리보기를 출시했습니다. 이는 AI가 실제로 컴퓨터를 제어할 수 있는 획기적인 단계입니다.

🖥️ Mac 컴퓨터 제어

Claude는 이제 인간의 사용자처럼 Mac 컴퓨터를 조작할 수 있습니다. 마우스 포인터 이동, 아이콘 클릭, 파일 편집 등 다양한 작업을 독립적으로 수행합니다.

⚙️ 통합 애플리케이션

Claude Code와 Claude 데스크톱 앱을 통해 더 광범위한 데스크톱 자동화를 지원합니다. 복잡한 작업들을 자동으로 처리할 수 있습니다.

Claude의 컴퓨터 제어 능력

🎯 지원하는 작업

  • 마우스 포인터 제어: 정확한 위치에 마우스 포인터를 이동하고 아이콘을 클릭할 수 있습니다
  • 파일 관리: 로컬 파일을 열고 편집할 수 있습니다
  • 스프레드시트 자동화: 복잡한 스프레드시트를 자동으로 작성하고 채울 수 있습니다
  • 전문 소프트웨어 작업: 전문가 수준의 소프트웨어 작업을 독립적으로 처리할 수 있습니다

지원 애플리케이션

Claude Code

코딩 작업을 지원하며, 개발자들이 더 효율적으로 작업할 수 있도록 도와줍니다.

Claude 데스크톱 앱

더 광범위한 데스크톱 자동화를 제공하며, 다양한 업무 작업을 처리할 수 있습니다.

Remote Control - Dispatch

모바일 기기에서 음성 또는 텍스트 명령을 보내는 동반 도구로, 크로스 디바이스 기능을 제공합니다.

의의 및 영향

AI 상호작용의 진화

이 기능은 텍스트 및 이미지 AI 상호작용에서 실제 데스크톱 워크플로우 통합으로의 획기적인 전환을 나타냅니다. 사용자들은 이제 AI를 "디지털 워커"로서 활용하여 반복적이고 복잡한 작업을 자동화할 수 있게 됩니다. 이는 업무 생산성을 크게 향상시킬 수 있는 잠재력을 가지고 있습니다.

향후 Windows 및 Linux 지원 확대, 더 많은 사용자층으로의 확산, 그리고 다양한 전문 소프트웨어와의 통합이 예상됩니다.
