개요

Zhipu(智谱)가 GLM-5.1 모델을 공식 발표했습니다. 모든 Coding Plan 사용자(Max, Pro, Lite)가 호출할 수 있으며, Claude Code와 OpenClaw 등 주요 개발 도구에서 바로 사용 가능합니다.

GLM-5.1은 이전 GLM-4.7의 후속 모델로, 추론 능력이 지원되며 텍스트 입력을 처리합니다.

설정 가이드

Claude Code 설정 (~/.claude/settings.json)
{
  "env": {
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "glm-5.1",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "glm-5.1",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "glm-4.5-air"
  }
}
OpenClaw 설정 (~/.openclaw/openclaw.json)
{
  "model": "glm-5.1",
  "context_window": 204800,
  "max_output": 131072,
  "reasoning": true,
  "cost": { "input": 0, "output": 0 },
  "input_types": ["text"]
}

주요 특징

🎯

전 플랜 사용 가능

Max, Pro, Lite 등 모든 Coding Plan 사용자가 GLM-5.1을 호출할 수 있습니다. 추가 비용 없이 최신 모델을 체험할 수 있습니다.

⚡

GLM-4.7에서 업그레이드

기존 설정에서 GLM-4.7을 GLM-5.1로 변경하기만 하면 바로 사용 가능합니다. Haiku 모델은 기존 glm-4.5-air를 그대로 유지합니다.
