Huawei 🤖 AI

화웨이 클라우드 마다오 신규 버전 출시: 다중 에이전트 협업과 전역 메모리로 AI 개발을 '시스템적 생산성'으로 이끌다

2026-04-28

개요

Huawei의 Cloud CodeArts(클라우드 코드아츠) AI 코딩 에이전트가 새로운 버전을 공식 출시했습니다. 이번 업데이트는 '시스템화된 생산성'을 핵심 가치로 삼아 개발 효율성, 협업 방식, 코드 품질 개선부터 Skill 축적에 이르기까지 다방면에서 전면적인 최적화를 이루었으며, 기업 수준의 복잡한 개발 요구사항을 충족시키도록 설계되었습니다.

개발 효율성 강화

새로운 버전에서는 여러 세션 병렬 처리와 AI 에이전트 자동 분배 기능이 추가되어 복잡한 작업을 수행하는 능력이 크게 향상되었습니다.

  • 다중 세션 병렬 처리: 개발자는 하나의 IDE 내에서 여러 작업 스레드를 동시에 열어 독립적으로 다양한 개발 업무를 진행할 수 있어, 다중 작업 처리 효율성이 높아집니다.
  • 에이전트 자동 분배: 주 에이전트가 복잡한 작업을 필요에 따라 세분화하고, 동적으로 하위 에이전트를 생성하여 협업을 수행합니다. 각 하위 에이전트는 컨텍스트(문맥)를 독립적으로 격리하여 특정 하위 작업에 집중할 수 있게 함으로써, AI가 더욱 길고 복잡한 개발 프로세스를 지원하며 정확도가 크게 향상되었습니다.

프로젝트 협업 및 지식 관리

이번 버전은 '선호도 기억(Preference Memory)' 기능과 프로젝트 단위의 지식 축적 시스템을 강화했습니다.

  • 개인 선호도 기억: 사용자는 대화를 통해 언어, 개발 프로세스, 코딩 표준 등 개인적인 선호 규칙을 지속적으로 설정하고 업데이트할 수 있으며, 이 설정이 프로젝트 전반에 걸쳐 글로벌하게 적용됩니다. 이를 통해 개발 습관을 한 번 설정하면 장기간 재사용할 수 있어 반복적인 커뮤니케이션 비용을 획기적으로 줄여줍니다.
  • 프로젝트 스마트 명함 기능: 기존 프로젝트를 위한 구조화된 설명 문서를 원클릭으로 생성합니다. 이 기능은 코드 저장소를 자동으로 깊이 분석하여 프로젝트 아키텍처, 초기화 방식, 개발 표준 등 핵심 정보를 축적하고, 이를 AI 실행 과정의 지속적인 참고 자료로 활용합니다. 결과적으로 전체 코드 컨텍스트에 대한 의존도를 줄이고 토큰 소모를 효과적으로 낮추면서도, 코드 생성의 정확성과 일관성을 높여 프로젝트 기존 규범에 더욱 부합하게 만듭니다.

코드 품질 검증

새로운 버전에서는 AI 코드 리뷰 기능이 추가되어 개발된 코드를 다각도로 점검할 수 있습니다.

  • 다차원 검사: 보안 취약점, 컴파일 오류, 요구사항 충족도 등 여러 측면에서 검사를 수행합니다.
  • 완벽한 보고서 제공 및 수정 지원: 문제 위치, 영향 설명, 심각도 레벨이 포함된 종합적인 리뷰 보고서를 생성하며, 자동 수정 기능까지 지원하여 코드 품질과 엔지니어링 적용 가능성을 높입니다.

생태계 확장 및 Skill 구축

Huawei는 20년 이상의 연구 개발 경험과 수천억 줄의 코드를 기반으로, 요구사항 관리, 시스템 설계, 개발 테스트부터 배포까지 전 과정을 아우르는 고빈도 시나리오 Skill을 CodeArts에 내장하여 즉시 사용 가능하도록 했습니다.

또한, 새로운 버전에서는 '스킬 커스터마이징 작업실'이 도입되어 스킬 생성기 및 스킬 테스터가 포함되었습니다. 이를 통해 개발자는 자신만의 전용 스킬을 빠르고 쉽게 만들고 테스트하며 재사용할 수 있게 되어, 개인과 팀의 경험을 재활용 가능한 자산으로 축적할 수 있습니다.

총평

이번 CodeArts 업데이트를 통해 Huawei는 AI 개발의 새로운 패러다임을 가속화하고 있습니다. 다중 에이전트 협업, 지속적인 기억 기능, 그리고 엔지니어링 표준화를 결합함으로써 제품은 AI 프로그래밍에 대한 이해도를 높이고 있으며, 소프트웨어 개발을 더욱 효율적이고 지능적인 시스템 생산성 단계로 이끌고 있습니다.
