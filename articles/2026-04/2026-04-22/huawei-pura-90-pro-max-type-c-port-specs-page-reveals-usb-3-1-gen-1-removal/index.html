Huawei 📱 스마트폰

화웨이 Pura 90/Pro/Max 전 모델, USB 3.1 Gen 1 사양 설명 삭제한 Type-C 포트 스펙 페이지 공개

2026-04-22

Pura 90 시리즈 출시 및 스펙 변화

IT之家에 따르면, Huawei가 최근 Pura 시리즈 전 제품군 신제품 발표회에서 Pura 90, Pura 90 Pro, Pura 90 Max 세 모델을 공개했습니다.

데이터 포트 사양 비교 분석

Pura 시리즈의 데이터 케이블 인터페이스(Data Cable Interface) 사양이 모델 및 세대별로 변화를 보이고 있어 구매 시 유의해야 할 중요한 기술적 변경 사항이 확인되었습니다.

  • 신규 Pura 90 시리즈 (Pura 90 / Pro / Max)
  • 데이터 케이블 인터페이스 사양은 'USB Type-C'로 명시되어 있습니다.

반면, 이전 세대 모델 및 기존 Mate 라인업의 사양은 다음과 같습니다.

  • Pura 80 Pro 시리즈 (Pro / Pro+ / Ultra)
    데이터 케이블 인터페이스는 'USB Type-C, USB 3.1 Gen 1, DP 1.2'를 지원합니다.
  • Pura 80 표준 모델
    데이터 케이블 인터페이스는 'USB Type-C, USB 2.0'으로 명시되어 있습니다.
  • 기존 Huawei Mate 80 전 라인업
    여전히 'USB Type-C, USB 3.1 Gen 1'을 지원합니다.

총평

이번 Pura 90 시리즈는 플래그십 모델임에도 불구하고 데이터 케이블 인터페이스 사양이 단순히 'USB Type-C'로만 표기되어 있어, 고속 전송 규격(예: USB 3.1 Gen 1)에 대한 구체적인 정보가 부족합니다. 따라서 사용자는 이전 세대 Pro 모델들이 지원했던 높은 대역폭의 데이터 전송 속도를 기대하기 어려울 수 있으므로, 구매 전에 반드시 공식 스펙을 재확인할 필요가 있습니다.
