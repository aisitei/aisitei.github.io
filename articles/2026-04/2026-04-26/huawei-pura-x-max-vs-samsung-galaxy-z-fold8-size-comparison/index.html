Huawei Pura X Max와 Galaxy Z Fold8 Wide 크기 비교

IT之家에 따르면, 정보 출처인 @i冰宇宙가 지난 4월 24일 웨이보(Weibo)를 통해 Huawei Pura X Max와 Samsung Galaxy Z Fold8 Wide 두 폴더블 스마트폰의 크기를 비교 분석한 내용을 공유했습니다.

주요 스펙 및 디자인 비교

웨이보 내용에 따르면, 두 기종은 전개 상태와 접힌 상태에서 각기 다른 치수를 보여줍니다. 특히 화면 비율과 전체적인 본체 폭의 차이가 눈에 띕니다.

  • Samsung Galaxy Z Fold8 Wide
    • 전개 상태 크기: 123.9 x 161.4 x 4.3mm
    • 접힌 상태 크기: 123.9 x 82.2 x 9.8mm
    • 화면 비율: 내부 화면은 4:3, 외부 화면은 4.7:3을 채택했습니다.
  • Huawei Pura X Max
    • 전개 상태 크기: 120 x 166.5 x 5.2mm
    • 접힌 상태 크기: 120 x 85 x 11.2mm

상세 치수 비교 분석

IT지카는 공식 데이터를 바탕으로 두 모델의 상세한 수치를 표 형태로 정리하여 제공했습니다.

  • Huawei Pura X Max
    • 가로(Horizontal): 166.5mm, 세로(Vertical): 120.0mm
    • 전개 두께: 5.2mm, 접힌 두께: 11.2mm
    • 접었을 때 본체 폭: 85mm
    • 외부 화면 비율: 4.4:3, 내부 화면 비율: 4.24:3
  • Samsung Galaxy Z Fold8 Wide
    • 가로(Horizontal): 161.4mm, 세로(Vertical): 123.9mm
    • 전개 두께: 4.3mm, 접힌 두께: 9.8mm
    • 접었을 때 본체 폭: 82.2mm
    • 외부 화면 비율: 4.7:3, 내부 화면 비율: 4:3

총평

이번 비교 분석은 두 플래그십 폴더블폰의 물리적 크기와 디자인 차이를 명확히 보여줍니다. Huawei Pura X Max가 전반적으로 더 큰 가로 길이와 약간 더 두꺼운 접힌 상태를 보이는 반면, Samsung Galaxy Z Fold8 Wide는 상대적으로 얇고 외부 화면 비율이 높은 것이 특징입니다. 사용자들은 이 데이터를 바탕으로 휴대성과 사용 편의성에 중점을 두고 모델을 선택할 수 있을 것입니다.
