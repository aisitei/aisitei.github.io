서비스 개요

Tencent Cloud가 "랍스터(龙虾)" 기억 서비스 — TencentDB Agent Memory를 정식 출시했습니다. 이 혁신적인 서비스는 OpenClaw에 강력한 장기 기억 기능을 보완하며, "무료 원클릭 활성화" 지원으로 누구나 쉽게 도입할 수 있습니다.

4단계 점진적 기억 아키텍처

L0

원시 대화

원시 정보를 전량 보존하여 완벽한 대화 이력을 유지합니다. 모든 상호작용의 맥락을 보존합니다.

L1

원자 기억

사실, 선호도, 핵심 제약을 자동으로 추출합니다. 중요한 정보만 효율적으로 추려냅니다.

L2

장면 분할

프로젝트별로 자동 클러스터링되어 정밀한 정보 리콜을 가능하게 합니다. 상황에 맞는 기억 활용이 가능합니다.

L3

사용자 프로필

사용자 프로파일을 형성하여 개별 사용 습관에 맞게 적응합니다. 개인화된 경험을 제공합니다.

Lighthouse 콘솔에서 활성화

Lighthouse 콘솔에 직접 접속하여 간단한 설정만으로 TencentDB Agent Memory를 활성화할 수 있습니다. 별도의 복잡한 구성이 필요 없습니다.

로컬 배포 명령어

로컬 환경에서 OpenClaw를 사용 중인 경우, 다음 명령어를 실행하여 Agent Memory 플러그인을 설치할 수 있습니다:

openclaw plugins install @tencentdb-agent-memory/memory-tencentdb
