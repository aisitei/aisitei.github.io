Claude Code 소스 코드 유출 — 1,900개 파일, 51.2만 줄 TypeScript 코드 공개

2026년 3월 31일
원본 링크: IT之家
실리콘밸리가 들썩였다. Claude Code의 하위 레벨 코드가 방금 "오픈소스"되었다! 1,900개 이상의 파일, 51.2만 줄의 코드가 모두 터져나왔다. Anthropic의 개발자 Chaofan Shou가 폭로한 바에 따르면, Claude Code 소스 코드가 npm 레지스트리의 맵 파일을 통해 실수로 유출되었다고 한다.

관련 이미지

핵심 아키텍처 분석

유출된 코드는 Claude Code가 React + Ink(터미널 UI)로 구축되었으며, Bun 런타임에서 실행된다는 것을 보여준다. 기본 아키텍처는 다음을 포함한다:

Tools (도구상자)

  • 40개 이상의 독립 모듈
  • 파일 읽기/쓰기 기능
  • Bash 명령 실행
  • 내장 LSP 프로토콜
  • 서브 에이전트 생성 능력

QueryEngine.ts

  • 추론 엔진 (4.6만 줄)
  • 추론 로직 처리
  • Token 계산
  • Chain-of-Thought 루프
  • 성능 최적화

협동 시스템

  • 다중 에이전트 조정
  • IDE 깊이 통합
  • 실시간 동기화
  • 워크플로우 관리
  • 상태 추적

발견된 숨겨진 기능

코드명 Kairos - 자율 수호 프로세스

  • 자동 시스템 모니터링 및 자체 치유
  • 실시간 코드 검증 및 최적화
  • 예측적 오류 감지
  • 자동 성능 조정

"전자 펫 시스템" (18개 물종)

  • 개발자 경험 향상 목표
  • 상호작용형 컴패니언 에이전트
  • 맥락 학습 및 적응
  • 사용자 행동 패턴 분석

"Undercover Mode" (위장 모드)

  • Anthropic 직원의 운영 흔적 자동 제거
  • 감사 로그 검증
  • 보안 프로토콜 강화
  • 데이터 프라이버시 보호

확산 및 영향

유출 후 GitHub 프로젝트의 스타 수는 짧은 시간에 5,000개를 돌파했다. 개발자 커뮤니티는 광범위하게 백업과 아카이빙을 진행 중이다. 이 사건은 대규모 언어 모델의 내부 구조와 개발 프로세스에 대한 중요한 통찰력을 제공하며, 오픈소스 AI 개발과 독점적 소프트웨어 보안 간의 논의를 촉발했다.
